    # 事件

    在Mirai.Net内部，定义了一系列的事件来对消息、群、好友、群成员以及Bot的行为进行监听。

    这份文档不会详细地给出每一个事件的成员列表，因为每一个事件的名字都代表了它的作用，如果你想了解某个事件的详细信息，请参考它的文档 (opens new window)。

    下面是一些简单的例子：

    # 监听群员禁言事件

    bot.EventReceived
    .OfType<MemberMutedEvent>()
    .Subscribe(receiver =>
    {
        Console.WriteLine($"{receiver.Member.Name} 被禁言了");
    });

    # 监听群全体禁言事件

    bot.EventReceived
    .OfType<GroupMutedAllEvent>()
    .Subscribe(receiver =>
    {
        Console.WriteLine($"{receiver.Group.Id} 被全员禁言。");
    });

    # 概述

    可以很明显地发现，在Mirai.Net中，每个事件监听器都需要调用EventReceived，然后再根据需求订阅事件。比如上面的两个例子，它们都是这样的形式：

    bot.EventReceived
    .OfType<具体事件类型>()
    .Subscribe(receiver =>
    {
        //具体的操作
    });

    因为EventReceived是一个IObservable<EventBase>类型的属性，这是基于Rx.NET (opens new window)的订阅模式。所以你可以使用LINQ来操作事件，比如（如果你不熟悉LINQ的用法，建议先去了解一下）：

    bot.EventReceived
    .OfType<具体事件类型>()
    .Where(receiver => receiver.Group.Id == 12345)
    .Subscribe(receiver =>
    {
        //具体的操作
    });

    你也可以定义无数个事件监听器，每个事件监听器都是一个独立的订阅器，这样你就可以在不同的地方订阅不同的事件:

    bot.EventReceived
    .OfType<GroupMutedAllEvent>()
    .Subscribe(receiver =>
    {
        //具体的操作
    });

bot.EventReceived
    .OfType<UnmutedEvent>()
    .Subscribe(receiver =>
    {
        //具体的操作
    });


bot.EventReceived
    .OfType<OnlineEvent>()
    .Subscribe(receiver =>
    {
        //具体的操作
    });

    # 事件一览

    这是一份Mirai.Net封装的事件与mirai-api-http的事件一览表，如果你想了解某个事件的详细信息，请参考它的文档 (opens new window)。

    # Bot自身事件

    • OnlineEvent : Bot登录成功 (opens new window)

    • OfflineEvent : Bot主动离线 (opens new window)

    • OfflineForceEvent : Bot被挤下线 (opens new window)

    • ReconnectedEvent : Bot主动重新登录 (opens new window)

    • DroppedEvent : Bot被服务器断开或因网络问题而掉线 (opens new window)

    # 好友事件

    • FriendInputStatusChangedEvent : 好友输入状态改变 (opens new window)

    • FriendNickChangedEvent : 好友昵称改变 (opens new window)

    • FriendRecalledEvent : 好友消息撤回 (opens new window)

    # 群事件

    • PermissionChangedEvent : Bot在群里的权限被改变. 操作人一定是群主 (opens new window)

    • MutedEvent : Bot被禁言 (opens new window)

    • UnmutedEvent : Bot被取消禁言 (opens new window)

    • JoinedEvent : Bot加入了一个新群 (opens new window)

    • LeftEvent : Bot主动退出一个群 (opens new window)

    • KickedEvent : Bot被踢出一个群 (opens new window)

    • GroupMessageRecalledEvent : 群消息撤回 (opens new window)

    • NudgeEvent : 戳一戳事件 (opens new window)

    • GroupNameChangedEvent : 某个群名改变 (opens new window)

    • GroupEntranceAnnouncementChangedEvent : 某群入群公告改变 (opens new window)

    • GroupMutedAllEvent : 是否全员禁言 (opens new window)

    • GroupAllowedAnonymousChatEvent : 是否允许匿名聊天 (opens new window)

    • GroupAllowedConfessTalkChanged : 坦白说 (opens new window)

    • GroupAllowedMemberInviteEvent : 允许群员邀请好友加群 (opens new window)

    • MemberJoinedEvent : 新人入群的事件 (opens new window)

    • MemberKickedEvent : 成员被踢出群（该成员不是Bot） (opens new window)

    • MemberLeftEvent : 成员主动离群（该成员不是Bot） (opens new window)

    • MemberCardChangedEvent : 群名片改动 (opens new window)

    • MemberTitleChangedEvent : 群头衔改动（只有群主有操作限权） (opens new window)

    • MemberPermissionChangedEvent : 成员权限改变的事件（该成员不是Bot） (opens new window)

    • MemberMutedEvent : 群成员被禁言事件（该成员不是Bot） (opens new window)

    • MemberUnmutedEvent : 群成员被取消禁言事件（该成员不是Bot） (opens new window)

    • MemberTitleChangedEvent : 群员称号改变 (opens new window)

    # 申请事件

    • NewFriendRequestedEvent : 添加好友申请 (opens new window)

    • NewMemberRequestedEvent : 用户入群申请（Bot需要有管理员权限） (opens new window)

    • NewInvitationRequestedEvent : Bot被邀请入群申请 (opens new window)
